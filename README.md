# ブックマーク管理AIエージェント

## 概要

Webブラウジング中にブックマークした記事を一元管理し、AIによる自動タグ付けと定期的なレコメンド通知により、情報のインプットと活用を効率化するシステムです。

## 主な機能

- **🔖 汎用ブックマーク登録**: 任意のWebページをChrome拡張機能から登録
- **🤖 AI自動タグ付け**: メタデータとユーザーメモからLLMが適切なタグを生成
- **📊 Notion一元管理**: すべてのブックマークをNotion Databaseで管理
- **📬 定期レコメンド**: 週1回、読み返すべき記事をSlackで通知

## 技術スタック

- **フロントエンド**: Chrome拡張機能 (Manifest V3)
- **バックエンド**: n8n (ワークフロー自動化)
  - ※このリポジトリには `n8n` のワークフロー定義は同梱していません（必要に応じて別途管理）
- **データベース**: Notion Database
- **AI処理**: OpenAI API (GPT-4o-mini)
- **通知**: Slack

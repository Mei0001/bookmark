# アイコンファイル

このディレクトリには、Chrome拡張機能のアイコン画像を配置します。

## 必要なファイル

以下のサイズのPNG画像を用意してください：

- `icon16.png` - 16x16px（ツールバー用）
- `icon48.png` - 48x48px（拡張機能管理画面用）
- `icon128.png` - 128x128px（Chrome Webストア用）

## デザインガイドライン

- シンプルで視認性の高いデザイン
- ブックマーク（本のアイコン）を連想させるデザイン推奨
- 背景は透過または白
- AIを連想させる要素を含めても良い

## アイコン作成ツール

- [Figma](https://www.figma.com/)
- [Canva](https://www.canva.com/)
- [favicon.io](https://favicon.io/)（簡易的なアイコン作成に便利）

## 一時的な代替案

開発初期段階では、以下のようなプレースホルダーテキストアイコンでも問題ありません：
- 📚（本の絵文字）をベースにしたシンプルなアイコン
- BMA（Bookmark AI Agentの略）の文字ベースのアイコン
